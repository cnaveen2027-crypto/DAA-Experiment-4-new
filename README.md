# 🚀 Dijkstra Shortest Path Visualizer

A simple **Flask web app** that demonstrates **Dijkstra's Algorithm** for shortest paths.  
The app lets you input a graph (vertices + weighted edges), choose a source vertex, and then:

- ✅ Displays shortest path distances in a table  
- ✅ Shows reconstructed paths from the source to each vertex  
- ✅ Generates a **graph visualization** highlighting shortest paths
